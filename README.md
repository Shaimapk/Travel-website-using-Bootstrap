Green Land Tours is a responsive travel website built using HTML, CSS, and Bootstrap 5. 
It showcases Kerala tour destinations and packages with a modern UI design. 
The project demonstrates the use of Bootstrap components such as navigation bars, carousels, cards, modals, and responsive grid layouts.

Features:
Responsive navigation bar with dropdown menu
Image carousel with booking modal
Destination cards with Bootstrap grid system
Tour packages section with styled cards
Trip booking and callback modals
Clean and modern responsive design

Technologies Used : HTML5, CSS3, Bootstrap 5
